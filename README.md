A premium, professional digital experience for a boutique coffee roaster.

## Features
- **Premium Frontend**: Responsive, elegant design without frameworks.
- **Dynamic Catalog**: Products managed via the admin panel.
- **Admin Panel**: Secure-ish area to manage products and prices.
- **Database**: SQLite for zero-config persistence.
- **API**: RESTful API serving products and handling uploads.

## Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the server:
   ```bash
   npm start
   ```

3. Open your browser:
   - Public Website: `http://localhost:3000`
   - Admin Panel: `http://localhost:3000/admin` (Login: `admin` / `admin123`)

## Structure
- `/public`: Frontend static files (HTML, CSS, JS)
- `/server`: Backend Logic (Node.js, Express, SQLite)
- `/uploads`: Directory for product images

## Technologies
- Node.js & Express
- SQLite
- Vanilla HTML/CSS/JS
